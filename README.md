# Arpit Patel Portfolio

Static React portfolio site for Arpit Patel, focused on Salesforce development experience, certifications, and project work.

## Local setup

1. `cd c:\\Users\\Arpit PATEL\\Desktop\\Arpit-Portfolio`
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:5173`

## Build

`npm run build`

## Preview production

`npm run preview`

## Deploy to Render (free static site)

1. Push this folder to GitHub.
2. Create a new static site on Render: https://dashboard.render.com/new/static
3. Connect your GitHub repo and choose `main` branch.
4. Set build command: `npm install && npm run build`
5. Set publish directory: `dist`
6. Click `Create Static Site`.

Once deployed, Render provides a URL like `https://<your-project>.onrender.com`.

## Optional: Deploy to Netlify

1. Create Netlify site -> connect repo.
2. Build command: `npm run build`, publish directory: `dist`.
